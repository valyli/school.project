00085 /*
00086  * The MemBufFormatTarget is a derivative from XMLFormatTarget, which user code
00087  * may plug into DOMWriter to retrieve the serialized XML stream (from DOM Tree)
00088  * in a memory buffer.
00089  *
00090  * The MemBufFormatTarget is initalized to have a memory buffer of 1023 upon
00091  * construction, which grows as needed. The buffer will be deleted when
00092  * MemBufFormatTarget is destructed; or will be reset when the reset() function
00093  * is called.
00094  *
00095  * The MemBufFormatTarget returns a NULL terminated XMLByte stream upon request,
00096  * through the method getRawBuffer(), and user should make its own copy of the
00097  * returned buffer if it intends to keep it independent on the state of the
00098  * MemBufFormatTarget.
00099  */
00100
