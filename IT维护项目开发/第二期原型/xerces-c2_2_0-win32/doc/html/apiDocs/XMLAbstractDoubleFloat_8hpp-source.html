00106 /***
00107  * 3.2.5.1 Lexical representation
00108  *
00109  *   double values have a lexical representation consisting of a mantissa followed,
00110  *   optionally, by the character "E" or "e", followed by an exponent.
00111  *
00112  *   The exponent ·must· be an integer.
00113  *   The mantissa must be a decimal number.
00114  *   The representations for exponent and mantissa must follow the lexical rules
00115  *   for integer and decimal.
00116  *
00117  *   If the "E" or "e" and the following exponent are omitted,
00118  *   an exponent value of 0 is assumed.
00119 ***/
00120 
00121 /***
00122  * 3.2.4.1 Lexical representation
00123  *
00124  *   float values have a lexical representation consisting of a mantissa followed,
00125  *   optionally, by the character "E" or "e", followed by an exponent.
00126  *
00127  *   The exponent ·must· be an integer.
00128  *   The mantissa must be a decimal number.
00129  *   The representations for exponent and mantissa must follow the lexical rules
00130  *   for integer and decimal.
00131  *
00132  *   If the "E" or "e" and the following exponent are omitted,
00133  *   an exponent value of 0 is assumed.
00134 ***/
