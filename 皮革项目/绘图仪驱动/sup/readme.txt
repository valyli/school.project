驱动程序注释：（适用于所有版本）
-------------------------------


网络安装
--------
如果您在服务器上运行 AutoCAD 版本而需要安装本驱动程序，需记住以下要求：

1.- 驱动程序可以在服务器上安装，并可从与其相连的任一台计算机上使用。

2.- 您需要对服务器硬盘驱动器目录的写入许可。
如果您没有此写入许可，系统管理员或其它具有正常访问权的计算机用户
则必须运行本安装。
    
3.- 在安装本驱动程序过程中，不能运行 AutoCAD 会话。因此在运行本安装程序时，
您可能需要请您的同事退出他们正在运行的 AutoCAD 会话。



文件传输错误
------------
安装本驱动程序时，如果收到“通用文件传输错误”信息，请确保您具有选定安装的目录和所有子目录的写入许可。

如果您准备将现有驱动程序升级为 4.0 版。
--------------------------------------
如果您的 AutoCAD 配置已设置为使用本驱动程序的较早版本（3.3 版或更早），
请注意，您需要在 AutoCAD 中重复绘图仪的配置过程，方可使用此 4.0 版的驱动程序。此外，安装此新版本时，将会删除本驱动程序的旧版本。

请参阅以下有关特定系统的附加注释。


取消打印作业
------------
您无法使用状态监视器来取消处于拼图作业中的图件；但在以后的拼图中，如果其中
包含数页取消的作业，则可加以取消。

多重 AutoCAD 会话
-----------------
虽然 AutoCAD 第 13 和 14 版允许您同时运行几个会话，但本驱动程序只能用于其中之一。

最小介质尺寸
------------
如果您指定的用户尺寸小于 A4（在 PLOT、HPMPLOT 或 HPRENDER 中），并且正在
使用卷筒介质，则图形将会按正确的尺寸绘制，但介质将按 A4 尺寸裁切。

AutoCAD 应用程序注释：

用于 Windows 的 AutoCAD 第 12 版用户。
-----------------------------------
1.- 	确定您已安装用于 Windows 的 AutoCAD 第 12 版的最新版本（C4 或更新版本）。
	Autodesk 可为所有用于 Windows 的 AutoCAD 用户提供免费升级至此版本的
	服务。

2.- 	为确保您正确安装用于 AutoCAD 的 HP 驱动程序，请检查以下文件的内容：

	文件 1：ACAD.ADS

	此文件可在默认目录 C:\ACADWIN 中找到，其中应包括下列命令：

	acadapp
	C:\ACADWIN\SUPPORT\HPMPLOT.EXEAutoCAD application notes:
	文件 2：WIN.INI

	此文件可在默认目录 C:\WINDOWS 中找到，在文件的

	[hpgl2acaddrv]

	段中，应包括下列一行语句：

	config＝C:\ACADWIN\

	或任何一个存放 AutoCAD 配置文件的目录。

3.- 	您只能在每台个人计算机上为一个 AutoCAD 第 12 版配置安装此 HP-GL/2 
	驱动程序。


用于 Windows 的 AutoCAD 第 13 版用户。
-------------------------------------

此版本需要特别大的内存容量。如果您在安装过程中遇到错误信息，我们建议您为
计算机添加内存。

（Autodesk 建议至少具有 16MB 的 RAM，但请留意，这是基本配置）


用于 Windows 的 AutoCAD 第 14 版用户。
------------------------------------

如果您收到驱动程序丢失或不一致的错误信息，您可用以下方式手动组织 acad14.cfg 文件：

1.- 确保您已退出 AutoCAD，且未运行此程序。

2.- 在选择的文本编辑器中打开 acad14.cfg 文件。
    acad14.cfg 文件是 ASCII 文件，因此几乎可以在所有的文本编辑器中阅读。

3.- 将出现的所有字符串

      "Hewlett-Packard HP-GL/2 devices, ADI 4.3 - for Autodesk by HP"

    替换为字符串

      "Hewlett-Packard HP-GL/2 devices v4.0, ADI 4.3 - by HP".

    这告诉 AutoCAD 与相关的设备配合使用时，要使用新的驱动程序。

4.- 保存文件并退出文本编辑器。

现在应能够启动 AutoCAD 而不会收到错误信息。

您可能要再次运行“配置”，并选定“修改”及“重新配置”以重新配置已安装的所有设备。请注意 HP LaserJet 现在支持光栅图象，因此您应重新配置 LaserJet 4 设备为可与本驱动程序配合使用的“HP LaserJet（11x17，A3）”或“HP LaserJet（标准）”。


Windows for Workgroups 用户。
----------------------------

Windows for Workgroups 用户，请检查您是否具有操作系统文件的最新版本。


Windows NT 用户
----------------
1.- 如果您已配置打印机使用指定的 LPTx 端口，并且您要使用此端口从 AutoCAD 中进行打印，您将收到一个错误信息，提示您 AutoCAD 无法设置此端口。这是因为 AutoCAD 尝试直接访问硬件。为避免发生此问题，请不要配置任何打印机使用您要从 AutoCAD 中进行打印时将要使用的 LPTx 端口。

3.- 在 Windows NT 系统中，您如何从 AutoCAD 第 13 版中执行绘图，是否通过连接在服务器上的绘图仪？

    a) 转到 DOS 命令提示符并键入：

	net use LPTx "网络绘图仪名称" /persistent:yes

    b) 用通常的方式在 AutoCAD 中配置绘图仪，在屏幕上询问您共有多少个 COM/LPT 端口时，请输入一个点号(.)表示没有。

    c) 在配置绘图仪时，请转至系统参数部分，将默认绘图文件名更改为 c:\<本地路径>\LPTx。


4.- 如果安装程序无法检测到以前安装的 AutoCAD...

    如果您在 Windows NT 4.0 中，且使用具有管理员优先权的帐号安装了 AutoCAD，则也应使用具有管理员优先权的帐号安装本驱动程序。否则，安装程序可能无法检测到 AutoCAD 安装的位置。您也可以在安装程序提示时，输入 AutoCAD 安装的确   切目录位置，以避开此问题。
